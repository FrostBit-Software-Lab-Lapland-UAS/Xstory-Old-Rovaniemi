-------------------- SUOMEKSI --------------------

Xstory-hanke
- Tämän repositorion lähdekoodi on kehitetty Xstory-hankkeessa
- Hankkeen kokonaisbudjetti oli 592 108 €. Hanketta osarahoitti
Euroopan aluekehitysrahasto / Lapin liitto. Myönnetty
EU- ja valtion rahoitus oli 473 682€.
Hankkeen kesto 01.07.2023 - 30.12.2025
- https://innovationlapland.com/xstory-hanke/
- https://www.frostbit.fi/portfolio-fi/xstory/
</br>

Tästä repositoriosta yleisesti
- Repositorio sisältää 3D-visualisoinnin Rovaniemen keskustasta sellaisena kuin se oli 1930-luvun alkupuolella
- Visualisointi perustuu arkistolähteisiin, mutta ei ole täydellisen tarkka johtuen sekä aukoista lähdemateriaalissa että projektin resursseista
</br>

Unreal Enginen versio
- Projektissa on käytössä Unreal Engine 5.5
</br>

Pluginit
- Jos Unreal Engine kehottaa ensimmäisellä käynnistyksellä enabloimaan plugineja, enabloi kaikki suositellut
</br>

Levelien rakenne
- Rakennukset on sijoitettu OldRovaniemi -levelille. Niitä voi muokata koskematta Core-leveliin
- Editorissa suorittaminen kannattaa tehdä Core-levelistä
</br>

Suositukset laitteistolle
- Kehittäessä on käytetty laitteita, joissa on vähintään 32GB RAM ja Nvidia 3080. Emme takaa projektin sujuvaa toimintaa tämän alittavalla laitteistolla.
